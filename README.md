# Prumyslovka Liberec - Školní liga stolního fotbálku (Kantýnová Liga PSLIB)

**Informační a sociální systém navržený pro kompletní organizaci a sledování celoškolní ligy ve stolním fotbálku.
Aplikace digitalizuje evidenci zápasů mezi jednotlivými třídami, automaticky generuje detailní statistiky týmů i jednotlivých hráčů a zároveň funguje jako
komunitní platforma, kde mohou studenti sledovat aktuální výsledky, plánovat nové zápasy a reagovat na herní výkony svých spolužáků.**
